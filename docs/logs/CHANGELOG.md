# LiveMines Simulator - 更新日誌 (Changelog)

## [v2.29] - 2026-05-13
### ✨ 新增與優化 (Features & Refactoring)
- **Agent Traffic 模式 UI 精煉**:
  - 將 `GameBoard` 上的「本局總成本」調整為「本局大盤總押注」，並移除負號與更改文字顏色，以符合大盤整體概念。
  - 隱藏人流模式下不必要的手動操作設定（預設單格金額、全選 9 格、購買閃電 checkbox、BONUS 闖關設定）。
- **左側邊欄「最新一局明細」更新**:
  - 將原本單機視角的算式明細，重構為大盤視角。
  - 在 Bonus 結算時，顯示 **「🌐 BONUS 世界線 (Public Result)」** 面板，包含各樓層的人流走向（Cashout/續闖/陣亡）。
  - 在 Base Game 明細中，僅列出有落球、被劈閃電、或是至少有一名 Agent 押注的格子，並清楚顯示該格的**總押注**與**總派彩**。
- **歷史紀錄 (HistoryModal) 數值修正**:
  - 修復右側點擊格子顯示投資名單時，Agent 的押注金額未加上「付費閃電 50% 稅金」的顯示問題，確保與左側大盤金額完美對齊。

### 🐛 錯誤修正 (Bug Fixes)
- 修復切換模式時，Persona 分佈因為清空歷史進度而跟著消失的問題。
- 修復剛載入網頁時，預設人流模式會因條件判斷錯誤而未能自動 fetch `agents.json` 的問題。
- 確保每次在「手動模式」與「人流模式」之間切換時，舊有資料會被自動清空，並重新產生該日的 Day Plan，避免不同邏輯的數據混雜。
